# Memory Game

Memory Game es un juego basado en el clásico juego de memoria en donde el jugador debe ir dando click a unas tarjetas e ir descubriendo las parejas, en este caso parejas de animales.

## Diseño

El diseño de este juego está relacionado con los juegos de plataforma minimalistas, en cual pretende tener la menor cantidad de colores para que las tarjetas de los animales sean los que realmente se llevan el protagonismo, las animaciones están creadas para llevar al jugador por una serie de pantallas y que sea todo un proceso muy agradable.

## Tecnologías

- [Greensock](https://greensock.com) : Es una herramienta muy poderosa de animación basada en javascript el cual permite la fácil y rápida implementación de diseño de interfaces y mejorar la interfaz de usuario.

- [React](https://reactjs.org/) : Herramienta para la creación de interfaces, el cual nos permite de una manera rápida y con mejor performance hacer una aplicación escalable y fácil de adaptar a diferentes ambientes.

## Available Scripts

En este proyecto lo primero que tienes que hacer es lo siguiente:

### 1 . Clonar este repositorio

Se debe clonar este proyecto accediendo en alguna parte de tu directorio y en la terminal hacer lo siguiente:

### `git clone https://github.com/andreslarrotta/memory-game.git`

Luego se deben instalar todas las dependencias para la correcta visualización del juego.

### `npm install`

Ya instaladas todas nuestras dependencias procedemos a compilar nuestro código con el siguiente comando.

### `npm start`

En el localhost:3000 se mostrará el ambiente de desarrollo el cual nos permitirá ver el funcionamiento del juego.
Esta página se actualizará cuando tu hagas algún cambio.
